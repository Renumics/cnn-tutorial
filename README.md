# CNN-Tutorial

## Install on Linux

Clone the repo

```bash
git clone https://github.com/Renumics/cnn-tutorial
```
Change the directory

```bash
cd cnn-tutorial
```
Create virtual env

```bash
python -m venv .venv
```
Active the virtual env

```bash
source .venv/bin/activate
```
Install dependencies

```bash
pip install -r requirements.txt
```

Start Jupyter Lab

```bash
jupyter lab
```

or use your preferred code editor!
